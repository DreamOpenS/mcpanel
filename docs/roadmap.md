# Roadmap

## Phase 1
- Core auth and RBAC
- Organization/team model
- Basic server management
- Node registration and health checks
- Dashboard UI and metrics

## Phase 2
- File manager
- Plugin and mod marketplace
- Scheduled backups
- Monitoring dashboards
- Audit and activity feed

## Phase 3
- Enterprise billing and quotas
- White-label branding
- Auto failover and load balancing
- AI analytics and recommendations

## Phase 4
- Global scaling and multi-region support
- Prometheus federation
- Advanced AI automation
- Plugin ecosystem extensions
